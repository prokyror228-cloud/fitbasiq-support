Политика конфиденциальности

Эта политика описывает, какие данные использует приложение FitBasiq, зачем они нужны и как пользователь может ими управлять.

Какие данные использует FitBasiq

FitBasiq позволяет сохранять:

  • email аккаунта, имя, цель тренировок, дату рождения, рост и текущий вес;
  • историю измерений веса;
  • упражнения, планы, расписание и настройки напоминаний;
  • тренировки, подходы, рабочий вес, повторы, заметки и оценку сложности;
  • локальные привязки QR-кодов к упражнениям.

Эти данные используются только для работы дневника тренировок, облачного восстановления и отображения личного прогресса.

Где хранятся данные

В демонстрационном режиме данные хранятся локально на устройстве. После создания настоящего аккаунта данные также синхронизируются с облачным хранилищем Supabase.

Доступ к облачным данным ограничен авторизацией и правилами Row Level Security, чтобы пользователи не могли видеть данные друг друга.

Камера и фотографии

FitBasiq запрашивает доступ к камере для сканирования QR-кодов и поиска похожего тренажёра по фотографии. Доступ к медиатеке используется только после выбора фото пользователем. Анализ изображения выполняется локально на устройстве, а фотографии не загружаются в облако FitBasiq.

Уведомления

FitBasiq может отправлять локальные напоминания о тренировках только после разрешения пользователя. Их можно изменить или полностью выключить в приложении и настройках iPhone.

Передача и продажа данных

FitBasiq не продаёт персональные данные. В текущей версии нет рекламы, рекламных SDK и рекламного отслеживания. Данные могут обрабатываться Supabase только для авторизации, хранения и синхронизации аккаунта.

Срок хранения и удаление

Облачные данные хранятся, пока существует аккаунт пользователя. Настоящий аккаунт и связанные с ним облачные данные можно безвозвратно удалить в разделе «Профиль → Аккаунт → Удалить аккаунт».

Локальные данные также можно удалить вместе с приложением. Некоторые сведения могут временно оставаться в защищённых резервных копиях инфраструктуры до их планового удаления.

Дети

FitBasiq не предназначен специально для детей и сознательно не собирает персональные данные детей без согласия законного представителя.

Изменения политики

При изменении способов обработки данных эта страница будет обновлена. Дата последнего обновления всегда указывается в начале документа.

Связь с поддержкой

По вопросам конфиденциальности и работы приложения используйте страницу поддержки FitBasiq. Не публикуйте в обращении личные данные.
