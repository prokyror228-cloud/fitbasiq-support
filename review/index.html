FitBasiq demo QR code

This QR code is provided for Apple App Review to test the local gym equipment QR-linking feature in FitBasiq.

QR value: FITBASIQ-DEMO-QR-001

How to test

  1. Open FitBasiq and create or sign in to a test account.
  2. Go to Profile -> QR тренажёров.
  3. Tap Сканировать QR-код and scan this QR code.
  4. If the QR code is not linked yet, select any exercise to link it.
  5. Scan the same QR code again to add the linked exercise to the active workout.

The QR feature is local to the user's account and does not upload photos or camera frames.
